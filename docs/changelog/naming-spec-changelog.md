# Naming Spec Changelog

## v1.0.0 — 2026-04-18

**Status**: stable

**Initial release** of mycodexvantaos naming spec v1.

### Introduced
- Three-layer naming model (Canonical, Composite, Protocol-Specific)
- 19 canonical capability identifiers
- Double-hyphen `--` as composite separator
- Full CI enforcement matrix (9 hard rules, 11 soft rules)
- Exception model with mandatory expiry dates
- Kubernetes namespace isolation for environment markers
- Complete regex table (13) as machine-readable validator input
